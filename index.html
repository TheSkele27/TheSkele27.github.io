Sunbeam Discord Bot

About:

The Subeam discord bot is a discord bot developed by TheSkele27#1337 with help from Matthew#0008. It uses the d.js module.

Default Prefix: ??

Features:

  • Fun commands
  • Moderation Commands
  • Report System           

Commands:

Command Example Default
addrole Adds a role to a member  
baguette Sends a French version of the Excuse me wtf meme  
ban Bans the tagged member  
botinfo Gives info about the bot  
cat Sends an image of a cat  
doggo Sends an image of a dog  
embed Sends text as an embed  
eval Owner only. Evaluates JavaScript commands  
file Starts the shop module  
game Displays the tagged users game  
help Sends the help command  
id Displays the tagged users ID  
info Information about the owner of the bot.  
kick Kicks the member  
members Organize the tags in a nice tree view.
